What Is HTML?

HTML(Hyper Text Mark-up Language) Is the programming language used to allow for the browser to determine what should be displayed on the page. HTML tells the DOM what content is trying to be displayed and provides details as to how it should be displayed and formatted based off of different attributes.

HTML - Tags

Tags are what the DOM is looking for as it’s the tags that tell the DOM how it should be displaying the content. For example if we wanted to display the page heading “Hello World” we could use an h1 tag.

" <h1>Hello World</h1>" - This is an HTML tag structure that mostly all tags will follow. As you can see there are two different tags that we use that look really similar, one is an opening tag and one is the closing tag.

The Opening "<h1>" tag tells the DOM that any content after this tag will be formatted as a heading, then we tell the DOM we have reached the end of the content that should be a heading by adding in the closing "</h1>" tag.

Now before we go any further lets talk more about HTML tags.

A tag can be made up of several different things, so far all we need to know is that these tags can have different attributes, each attribute allows the element to behave in a different way, for example we could add the title attribute to the <h1></h1> tag.



    

  1. <h1 title="Hello World">My Title</h1>
    2.

as you can see just after h1 inside the opening tag I have added in the title attribute, the title attribute tells the DOM that when we move our mouse over the text displayed it will also display a small tooltip with the message entered between the quotation marks.

This is a comment tag, these are used by developers; for developers by adding in meaningful comment statements that allow us to describe what different sections of our code are doing, these become really useful when you are working on large scale projects with other developers so that no one is left confused by the code written as we each write code in our own special ways.

Each tag seen above has a special purpose in how the website is displayed, the opening ‘html’ tag let’s the DOM know that we are going to be providing it HTML code, the ‘head’ tag is where we will be initialising different files later on in the project, the head section is one of the first that the DOM reads so it is always good practice to add links to any external resources needed for the project to run in this section.

The ‘Body’ tag as you might have guessed is where we are going to put in all of our main content, for the most part any code we add in to the HTML file will most likely always be between the body tags, after the body tag we reach the rest of the closing tags to let the DOM know that we are now finished providing it with HTML code.

What is CSS?

CSS(Cascading StyleSheets) is a stylesheet language that is used to describe to the DOM how the presentation of an HTML document should be.

An easy way to think of HTML and CSS together is like a face. HTML is used to describe the different features present, once all the features are in place, CSS acts as the makeup to the face. CSS can be used to style the HTML document, for example CSS is responsible for changing different characteristics of HTML elements such as the style of the font, colour of the font, etc.

CSS Syntax - CSS rules are made up of two things; a selector and a declaration block. The selector is simply the elements name, class, or ID, it's declaration needs a property which in turn requires a value. The syntax looks a bit like this:
h1{color:red; font-size:18px;}

CSS - Selectors

Their are 5 main categories of selectors, but for now we will focus on using the Simple selectors.

CSS Selectors are used to find or target HTML elements that you would like to style. The most basic and most generic way of selecting an element is to just simply use the name of the element.

Another way that we might like to use for selecting the element would be by giving it an ID. To do this we give the tag an ID aswell as or instead of a class. This looks like:

It is always good practice to use ID's to uniquely identify an element and classes to group elements with similar properties together.
