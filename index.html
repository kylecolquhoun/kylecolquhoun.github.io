What Is Python?

Python, first released in 1991, is one of the most popular languages at the time and is used by numerous developers worldwide. Python is considered to be a high-level language that prides it’s self on its easy to read code due to the indentation procedures that you follow when writing blocks of code.

What Is Flask?

Flask is considered a micro web framework due to it not requiring on a particular set of tools or libraries. Due to this it has a strong community that provide different tools and libraries for you to make use of in your development, this could be for anything from setting up a database; to performing input validation.

Flask was written / developed in Python without a Database abstraction layer making it lightweight and easy to use compared to one of it’s close buddies Django.

Django is still a good choice but is a large-scale framework that is great for database-driven websites.

For the time being we will be using Flask.

First - The basics!

I’m sure by now you at least know what a website is and what their most common uses are. We use them constantly and these days even shops have decided that they are best to jump on the trend and join the ever-growing web community to maximise their client reach.

As a developer. Websites can mean a lot more than just another place to order and buy a nice new shiny watch, it’s a vision, a sense of achievement, it’s one way you can get your idea out their to the world. And the best part? It’s not as hard as you’d think.

Websites consist of 3 main web technologies. These are HTML5, CSS3 and JS.

HTML(Hyper Text Markup Language) is the most commonly used web technology. It is used to describe to the DOM(document Object Model) what type of content is to be displayed on the website and tells it how to display it, HTML tells the DOM this by defining what is called “tags”.

Below you can see a basic HTML page structure! 



    

  1. <html>
    2.

    3. 

  3.       <title>My First Site</title>
    4. 

  4.     </head>
    5. 

  5.   <body>
    6. 

  6. <!-- Code Goes Here -->
    7. 

  7. 

  8. <style>
    9. 

  9. </style>
    10. 

  10.   </body>
    11. 

  11. </html>
    12.

lets work our way down the tags!

  • <html></html> - These tags are used to specify that all data within these tags should be read, compiled and displayed by the DOM.

  • <head></head> - These Tags are one of the first interpreted by the DOM and is good practice to add in any files that may need loaded in here, such as CSS files, or script files to allow them to load at the start of the code instead of the end as this could cause issues in the future.

  • <title></title> - These tags are used to specify what the name of your website is going to be, the title tags are always located in between the ‘<head></head>’ tags in the document as this gives your site an identity.

  • <body></body> - These tags are used to specify where the main content of the website is, in between these tags you will have different tags representing different content such as text and multimedia items that you want to display to the user.

    • Checkout w3schools for a list of more tags and how to use them.

What Is JavaScript?

JavaScript is an object-oriented scripting language and like python is also considered a high-level language. JavaScript is the backbone of a website, it allows you to dynamically update the content on the DOM by using triggering certain events.

JavaScript is a useful tool for bringing a website to life as it adds functionality to a website and can be used for a variety of different things, from form validation to animating a day-night circle animation on your webpage.

For this reason javascript is a great language to use and in this tutorial we will eventually learn how we can pass data between JavaScript and Python by using a text format known as JSON(JavaScript Object Notation) to send data to and from each of the languages.
